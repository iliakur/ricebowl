# Misc Configurations

## Software requirements
- i3 (including i3bar)
- py3status
- zsh
- meld
- vim


# Setup Steps

## Workrave
```
ln -s /path/to/configs/workrave.ini ~/.workrave/
ln -s /path/to/configs/workravectl.sh ~/bin/workravectl
```

## Spacemacs ##
Configuration lives in `.spacemacs` file and the `private` folder.
